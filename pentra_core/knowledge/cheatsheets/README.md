# Pentra Structured Cheat-Sheet Registry

This directory stores structured cheat-sheet intelligence used to prepare
Phase 3 capability packs.

It does **not** store bulk copied cheat-sheet bodies.

Instead it stores:

- categorized source selections
- provenance-tagged references
- ontology-family mappings
- Phase 3 capability-pack mappings
- target-profile mappings
- payload inclusion policy

Design rules:

- official public or official project sources must anchor every category
- community public payload collections may widen coverage, but only as
  supplemental entries
- this layer stores metadata and short structured notes, not giant payload
  dumps
- runtime proof remains the only authority for real findings
